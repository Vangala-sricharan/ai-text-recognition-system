# AI Text Recognition System

## 📌 Overview

This project is an AI-based Optical Character Recognition (OCR) system developed using Python. It extracts text from images using OpenCV for image processing and Tesseract OCR for text recognition.

---

## 🚀 Features

- Extract text from images
- Image preprocessing using OpenCV
- Optical Character Recognition (OCR)
- Save extracted text to a text file
- Simple and beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python
- OpenCV
- Tesseract OCR
- pytesseract

---

## 📂 Project Structure

```
AI-Text-Recognition-System/
│
├── images/
│   └── sample.png
├── output/

├── app.py
├── requirements.txt
└── README.md
```

---

## ⚙️ Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

Install **Tesseract OCR** and update the path in `app.py` if required.

---

## ▶️ Run the Project

```bash
python app.py
```

---

## 📄 Output

- Reads text from an image.
- Displays the extracted text in the terminal.
- Saves the extracted text to:

```
output/extracted_text.txt
```

---

## 📸 Sample

Input:
- Image containing printed text

Output:
- Extracted text displayed in the console
- Text saved as a `.txt` file
